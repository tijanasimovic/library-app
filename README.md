# # Library Management Application

This is a simple C# desktop application developed as a project for managing a small library system. It allows users to manage books and track basic library operations.

## Features

- Add, edit, and delete books
- View list of available books
- Search for books by title or author
- Simple user interface for easy navigation

## Technologies

- **Programming Language:** C#  
- **Framework:** .NET (Windows Forms) 
- **Database:** SQL Server
- **Tools:** Visual Studio 2022
